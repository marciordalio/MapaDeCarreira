Sobre mim

Sou o Márcio, 18 anos, de São Paulo. A tecnologia sempre despertou minha curiosidade, e hoje sigo transformando ideias em soluções úteis, com atenção aos detalhes e foco nas necessidades das pessoas. Estudo no Instituto PROA, onde desenvolvo habilidades em front-end e back-end com React, Node.js, Java e MySQL. Também atuo como front-end e designer UX/UI na Braille Way. Fora do código, gosto de andar de bicicleta, jogar basquete, ir à praia, tocar pandeiro e, principalmente, aprender com quem está ao meu redor. Acredito que crescer é trocar experiências e estar aberto a novos olhares. Busco uma oportunidade como Dev Júnior ou Machine Learning Júnior para seguir evoluindo e contribuindo com dedicação.

Conhecimentos

HTML5

HTML5 é uma linguagem de marcação de hipertexto, que usamos para marcar os elementos da página e construí-la.

CSS3

CSS3 é uma linguagem de estilo usada para descrever a apresentação dos elementos da página os estilizando.

JavaScript

JavaScript é uma linguagem de programação usada para criar funcionalidades e comportamentos interativos em páginas web.

React

React é uma biblioteca JavaScript criada pelo Facebook, usada para construir interfaces de usuário dinâmicas, especialmente em aplicações de página única (SPA).

node.js

Node.js é um ambiente que permite usar JavaScript no servidor, ideal para criar APIs, servidores e aplicações em tempo real de forma rápida e escalável.

MySQL

MySQL é um sistema de gerenciamento de banco de dados relacional que usamos para armazenar, organizar e acessar dados de forma eficiente em aplicações web e sistemas diversos.

Java

Java é uma linguagem orientada a objetos usada para criar aplicações robustas, seguras e multiplataforma, como sistemas corporativos e apps Android.

Git

Git é um sistema de controle de versão que gerencia o histórico do código, facilitando o trabalho em equipe, a colaboração e a reversão de mudanças.

Photoshopp

Photoshop é um software da Adobe usado para editar, retocar e criar imagens e designs gráficos profissionais.

Canva

Canva é uma ferramenta online fácil de usar para criar designs visuais rapidamente, ideal para apresentações, posts, cartões, currículos e outros materiais.

Figma

Figma é uma ferramenta na nuvem para design e prototipagem de interfaces, que permite colaboração em tempo real para criar layouts modernos e funcionais.

Tailwind CSS

Tailwind CSS é um framework utilitário que facilita estilizar interfaces aplicando classes no HTML para criar layouts responsivos e personalizados rapidamente.
